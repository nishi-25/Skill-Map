# スキルマップ

チーム・組織のスキルを可視化・管理する Web アプリケーション。

## 主な機能

### ダッシュボード

- **全体モード** — 組織全体のスキル数・カテゴリー数・平均レベル・レベル分布をサマリー表示
- **グループモード** — グループ単位でのスキル集計・メンバーランキング
- **個人モード** — 特定ユーザーのスキル詳細・レーダーチャート
- **レーダーチャート / レベル分布グラフ** — Chart.js によるインタラクティブな可視化
- **ティア別プログレス** — スキル難易度ごとの達成状況
- **成長速度ランキング** — 直近30日間のレベルアップ数ランキング

### 分析

- **スキルギャップ分析**（グループモード） — グループ必須スキル vs メンバー習得状況のヒートマップ
- **スキルカバレッジ分析**（全体モード） — 属人化リスク・空白スキルの検出、カテゴリー別カバー率

### スキル管理

- **スキルカタログ** — 組織共通のスキルマスターを Admin / Manager が管理
- **スキル申告** — ユーザーが自身のスキルレベル（5段階）を申告
- **承認フロー** — Admin / Manager がスキル申告を承認・却下（コメント付き）
- **スキル履歴** — レベル変更履歴の自動記録
- **カテゴリー管理** — スキルの分類（カラーラベル付き）
- **ティア（難易度）** — カスタマイズ可能なスキル難易度ランク

### グループ管理

- **グループ作成・編集** — 階層型グループ構造（親グループ対応）
- **メンバー管理** — グループへのユーザー割り当て
- **グループスキル** — グループに必須スキルを割り当て（ギャップ分析に連動）
- **マネージャー** — グループごとの管理者設定

### ユーザー管理

- **ログイン認証** — セッションベース認証（署名付き Cookie）
- **ユーザー登録・承認フロー** — 新規登録後に Admin / Manager が承認
- **パスワードリセット** — メール送信によるパスワード回復
- **プロフィール** — 表示名・アバター画像のアップロード
- **権限管理** — Admin / Manager / User の3ロール

### 管理機能（Admin）

- **ユーザー管理** — 権限変更・アカウント有効化/無効化
- **メール設定** — GUI からの SMTP 設定（テスト送信対応）
- **ティア名カスタマイズ** — スキル難易度の名称変更

### その他

- **マニュアル** — ユーザー向け利用マニュアル（`/manual`）、管理者マニュアル（`/manual/admin`）
- **メール通知** — 承認リクエスト・ステータス変更・パスワードリセットの通知

## 起動方法（Docker）

```bash
# ビルド & 起動
docker compose up -d

# ブラウザで開く
open http://localhost:8190
```

初回起動時は `/setup` にリダイレクトされ、管理者アカウントを作成します。

### 環境変数

| 変数         | 既定値                    | 説明                                       |
| ------------ | ------------------------- | ------------------------------------------ |
| `SECRET_KEY` | `change-me-in-production` | セッション署名キー（本番運用時は必ず変更） |
| `APP_URL`    | `http://localhost:8190`   | アプリケーションの公開 URL                 |

メール（SMTP）設定は管理画面（`/admin/mail-settings`）から GUI で行えます。

## ローカル起動（開発用）

```bash
cd app
pip install -r requirements.txt
uvicorn main:app --reload --port 8190
```

## 技術スタック

- **Backend**: Python / FastAPI + SQLAlchemy（SQLite）
- **Frontend**: Bootstrap 5.3 + Bootstrap Icons 1.11 + Chart.js 4.4
- **認証**: itsdangerous（署名付きセッション Cookie）+ bcrypt
- **コンテナ**: Docker / Docker Compose
- **Python**: 3.12
