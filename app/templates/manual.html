システム概要

「スキルマップ」は、チームメンバーのスキルレベルを記録・可視化・管理するWebアプリケーションです。メンバーが自分のスキルを申告し、上長が承認することでチーム全体のスキルマップが形成されます。

ログイン・初期設定

新規登録

1ログイン画面の「新規登録」ボタンをクリック
2ユーザー名・メールアドレス・パスワードを入力して送信
3管理者（Admin）が承認するまでログインできません。承認後にメール通知が届きます（メール設定が有効な場合）
ログイン画面の「デモを確認する」ボタンでログインなしにサンプルデータを閲覧できます。

パスワードを忘れた場合

1ログイン画面の「ユーザー名・パスワードをお忘れの方」をクリック
2登録済みメールアドレスを入力するとリセット用リンクが届きます

ダッシュボード User Manager Admin

スキル状況の概要を確認する画面です。ロールによって表示内容が変わります。

フィルター

上部のドロップダウンでユーザーまたはグループを選択して絞り込めます。URLにパラメータとして保存されるためブックマークも可能です。

表示内容（全体モード）

  • 統計カード（対象メンバー・総申告スキル・平均レベル・カタログ総数）— カーソルを乗せると詳細が表示されます
  • グループ別サマリー — 各グループの平均レベルとクリックでグループビューへ
  • ティア別進捗 — 難易度（初級〜上級）ごとの達成率をリングで表示
  • スキル成長トレンド — 過去12週のレベルUP件数（週次）をグラフ表示
  • メンバーランキング — 平均レベル順のランキング表
  • 成長速度ランキング — 直近30日でレベルアップが多いメンバー
CSVエクスポートボタンでスキルマトリクスをダウンロードできます。

プロフィール・表示設定 User Manager Admin

サイドバーの名前エリアをクリックしてプロフィールページへ移動します。

アバター設定

プロフィール画像の右下カメラアイコンをクリックしてJPG・PNG・GIFなど（最大2MB）をアップロードできます。

表示設定

  • カラーテーマ — Warm（オレンジ）/ Cool（インディゴ）を切り替えられます。設定はブラウザに保存されます。
  • 言語 — 日本語 / English を切り替えられます。ナビゲーションメニューやUIラベルが翻訳されます。
サイドバー下部のトグルスイッチでダークモードの切り替えもできます。

グローバル検索 User Manager Admin

サイドバー上部の検索バーに文字を入力するとスキル名・メンバー名・グループ名を横断してリアルタイム検索できます。入力から0.2秒後に候補が表示されます。Escキーで候補を閉じます。

問い合わせ・要望 User Manager Admin

新規作成（User/Manager）

1画面右下のオレンジ色チャットアイコンをクリック
2「新しい問い合わせ」ボタンをクリック
3種別（問い合わせ/要望）・優先度・タイトル・詳細を入力して「送信する」
4Adminが返信すると赤いバッジで通知されます

Adminの対応

サイドバーの「問い合わせ・要望」ページで全チケットを管理できます。未対応・対応中のチケットは赤く強調表示されます。ステータスを「解決済み」か「クローズ」にするとアラートが消えます。

チャット画面は5秒ごとに自動更新されます。ブラウザのリロードは不要です。

要望の管理

種別を「要望」にすると、管理ページ上部の「要望リスト」に優先度順でピン留め表示されます。忘れずに対応できます。

スキルマップ（自己申告） User Manager Admin

スキルの申告手順

1サイドバーの「スキルマップ」をクリック — 難易度ティア（初級・基礎・中級・上級）のカードが表示されます
2ティアカードをクリックしてスキル一覧を開く。または「全スキルを一覧表示」で全ティアを一画面に表示
3各スキルの●ドットをクリック、またはプルダウンでレベルを選択
4【一般Userの場合】承認者（Manager/Admin）を選択して「申請する」をクリック
5【Admin/Managerの場合】選択後に即時自動承認されます

レベル定義

0未経験該当スキルの経験なし
1入門概念・基礎知識を学習中
2基礎基本的な操作・作業ができる
3実務可一人で業務に活用できる
4エキスパート他者への指導・高度な設計ができる
カテゴリー・グループでスキルを絞り込めます。スキル名で検索も可能です。

自分の申請状況 User

申告したスキルの承認状況を一覧で確認できます。

  • 承認待ち — 承認者のレビュー待ち
  • 承認済み — スキルが正式に登録されました
  • 差し戻し — 承認者からコメントがついています。内容を確認して再申請しましょう
差し戻された場合は承認者のコメントを確認し、必要に応じてレベルを修正して再申請してください。

成長タイムライン User

承認されたスキルの変化を時系列グラフで確認できます。いつどのスキルがレベルアップしたかを振り返ることができます。定期的に確認して自分の成長を実感しましょう。

教育リソース User Manager Admin

Manager/Adminが登録した学習サイト・教材へのリンク集です。カテゴリー別にグループ化されています。

  • キーワード検索・カテゴリーフィルター・スキルフィルターで絞り込めます
  • 「サイトを開く」ボタンで新しいタブに外部リンクが開きます
  • 各カードにカテゴリーバッジと関連スキルが表示されます

承認依頼 Manager Admin

個別承認・差し戻し

1「承認」ボタンをクリック — 任意でコメントを入力して承認
2「差し戻し」ボタンをクリック → 理由をインライン入力 → 「送信」で差し戻し

一括承認・差し戻し

1各行の左端チェックボックスで複数選択（ヘッダーの「全選択」で一括選択）
2「選択を承認」または「選択を差し戻し」ボタンをクリック

フィルター

申請者名で絞り込み・カテゴリーでの絞り込みが可能です。件数バッジで未処理数を確認できます。

スキルマトリクス Manager Admin

メンバー × スキルのヒートマップです。暖色（黄→橙→赤）ほど理解度が高いことを示します。

分析チャート

「分析チャートを表示」ボタンをクリックすると以下のグラフが表示されます。

  • 成長トレンド（月別） — 月ごとの平均レベル変化
  • レベル分布 — ドーナツグラフ
  • カテゴリー別レーダー — ユーザー切替可能
  • ユーザー別平均レベル — 横棒グラフ

ソート機能

テーブルヘッダーをクリックすると昇降順ソートができます。複数条件ソートも対応しています。カテゴリー昇順→難易度昇順のように優先順位をつけて並べ替えられます。

スキルカタログ管理 Manager（担当グループのみ） Admin（全体）

スキルの追加

1「スキルを追加」ボタンをクリック
2スキル名・カテゴリー・難易度ティア・説明を入力して保存

一括インポート（CSV/Excel/Markdown/JSON）

1「CSVインポート」ボタンをクリック
2形式タブ（CSV/Excel/Markdown/JSON）を選択
3「テンプレート」ボタンでサンプルファイルをダウンロード
4テンプレートを編集してアップロード → 「インポート実行」

絞り込み・ソート

キーワード・カテゴリー・ティア・アーカイブ状態でフィルタリングできます。テーブルヘッダークリックで複数条件ソートが可能です。

一括操作

チェックボックスで複数スキルを選択し、「アーカイブ」「カテゴリ変更」「難易度変更」を一括実行できます。

Adminのみ「カタログを初期化」ボタンでHILS向けデフォルトカタログに戻せます（既存のスキル申告データも削除されます）。

カテゴリー管理 Manager Admin

スキルを分類するカテゴリーの追加・編集・削除ができます。

  • カードビュー — カラーカードで各カテゴリーのスキル一覧を確認
  • 行（リスト）ビュー — テーブル形式でコンパクトに一覧表示

右上のビュー切替ボタン（グリッド/リスト）で表示を切り替えられます。設定はブラウザに保存されます。

カテゴリーを削除すると、そのカテゴリーに属するスキルは「未分類」になります。

グループ管理 Manager（担当のみ） Admin（全体）

グループ作成

1「グループ作成」ボタンをクリック
2グループ名と説明を入力
3担当Managerを検索欄で検索して複数選択（追加済みManagerは下部に表示）
4親グループを設定すると階層構造になり、親のスキルが継承されます
5「対象スキル割当」で必須スキルを設定するとギャップ分析が有効になります

メンバー追加

1グループカードの「メンバー管理」ボタンをクリック
2ページ上部の「メンバーを追加」ボタン → モーダルで複数選択して一括追加

表示ビュー

「カード」「ツリー」ボタンで表示を切り替えられます。ツリービューでは親子階層が視覚的に確認できます。

担当Manager検索欄でManagerを複数設定できます。設定したManagerだけが担当グループを管理できます。

教育リソース管理 Manager Admin

リソースの追加

1「リンクを追加」ボタンをクリック
2タイトル・URL（必須）、説明を入力 — URLを入力するとファビコンが自動表示されます
3カテゴリーを選択するとスキルの候補が絞り込まれます。関連スキルを指定すると検索しやすくなります
4「追加する」をクリックして保存

管理パネル Admin

システム全体のサマリーと新規登録ユーザーの承認待ち一覧が表示されます。

新しいユーザーが登録するとここに表示されます。承認するとそのユーザーがログインできるようになります。

ユーザー管理 Admin

  • 承認/却下 — 新規登録ユーザーを承認・却下できます
  • ロール変更 — user / manager / admin の3段階でロールを変更できます
  • パスワードリセット — Adminがユーザーのパスワードをリセットできます
  • アカウント削除 — ユーザーと関連データを削除します（操作は取り消せません）
ロールを変更するとアクセスできる機能が即座に変わります。慎重に操作してください。

メール設定 Admin

SMTPサーバーを設定することで、以下のメール通知が自動送信されます。

  • ユーザー登録承認通知
  • スキル承認・差し戻し通知
  • パスワードリセットリンク

設定手順

1SMTPホスト・ポート・ユーザー名・パスワードを入力
2「設定を保存」後、「テスト送信」で動作確認
Gmailの場合はアプリパスワードを使用してください。ポートはTLSなら587、SSLなら465が一般的です。

カタログ初期化 Admin

スキルカタログページの「カタログを初期化」ボタンで、現在のカテゴリ・スキル・スキル申告データをすべて削除し、HILS向けデフォルトカタログ（10カテゴリ・73スキル）に戻します。

この操作はスキル申告データも削除されます。実行前にCSVエクスポートでバックアップを取ることを推奨します。

クイックスタート

初回ログイン後にロール別のガイドツアーが自動起動します。サイドバー下部の「クイックスタート」ボタンからいつでも再実行できます。
