ユーザー管理 Admin

「管理」→「ユーザー管理」から、全ユーザーの管理を行います。

新規ユーザーの承認

1ユーザー管理ページで「未承認」のユーザーを確認します。バッジに承認待ち件数が表示されます。
2「承認」ボタンをクリックしてユーザーを有効化します。

ロール変更

1ユーザー一覧でロールを変更したいユーザーを見つけます。
2「Manager に変更」または「User に変更」ボタンでロールを切り替えます。user / manager / admin の3段階で変更できます。
注意: ロールを変更するとアクセスできる機能が即座に変わります。慎重に操作してください。

アカウント削除

ユーザーと関連データを削除します（操作は取り消せません）。

パスワードリセット

メール経由のリセット（推奨）

メール設定が完了していて、ユーザーがメールアドレスを登録している場合は、ログイン画面の「パスワードをお忘れの方」からリセット可能です。画面からAdminが直接「パスワードリセット」「仮パスワード発行」を行うこともできます。

管理者による手動リセット（Dockerコマンド）

メールが使えない場合やメールアドレス未登録のユーザーに対して、Dockerコンテナ内でスクリプトを実行してリセットします。

1サーバーに SSH でアクセスします。
2以下のコマンドでリセットスクリプトを実行します。
docker exec -it skillmap python -c "
import database, models, auth
db = next(database.get_db())
user = db.query(models.User).filter(models.User.username == 'ユーザー名').first()
if user:
    user.password_hash = auth.hash_password('新しいパスワード')
    db.commit()
    print(f'{user.username} のパスワードをリセットしました')
else:
    print('ユーザーが見つかりません')
"
3ユーザー名 と 新しいパスワード を対象ユーザーのものに置き換えてください。
重要: パスワードは6文字以上で設定してください。リセット後、ユーザーにパスワードを安全な方法で通知してください。

Admin パスワードのリセット

Admin アカウントのパスワードを忘れた場合も同様にリセットできます。

docker exec -it skillmap python -c "
import database, models, auth
db = next(database.get_db())
admin = db.query(models.User).filter(models.User.role == 'admin').first()
if admin:
    admin.password_hash = auth.hash_password('新しいAdminパスワード')
    db.commit()
    print(f'Admin ({admin.username}) のパスワードをリセットしました')
"

全ユーザーの一覧確認

docker exec -it skillmap python -c "
import database, models
db = next(database.get_db())
for u in db.query(models.User).all():
    print(f'ID:{u.id}  {u.username}  role={u.role}  approved={u.is_approved}  email={u.email or \"未設定\"} ')
"
