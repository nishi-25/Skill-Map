APIリファレンス

JSON形式で利用できる /api/* エンドポイントのリファレンスです。各エンドポイントの説明・必要なロール・Pythonサンプルコードを掲載しています。

認証モデル: このAPIはブラウザのセッションCookieで認証します。APIキーやBearerトークンはありません。Pythonから呼び出す場合は requests.Session() でログインし、同じセッションで以降のリクエストを送ってください。
動作確認（Try it out）: 各エンドポイントの「ここで試す」リンクをクリックすると、FastAPI標準のSwagger UI（/docs）の該当オペレーションが新しいタブで開きます。ブラウザでこのアプリにログイン済みであれば、Swagger UI側で「Authorize」操作をしなくてもセッションCookieが自動的に送信され、「Try it out」→「Execute」で実際にAPIを実行できます。

Search

MethodPath必要ロール説明
GET/api/search全ロールスキル名・メンバー名・グループ名を横断検索 ここで試す
{{ m.python_sample('import requests BASE_URL = "http://localhost:8190" session = requests.Session() session.post(f"{BASE_URL}/login", data={"username": "...", "password": "..."}) resp = session.get(f"{BASE_URL}/api/search", params={"q": "Python"}) print(resp.json()) # => {"results": [{"type": "skill", "label": "Python", ...}, ...]}') }}

Dashboard

MethodPath必要ロール説明
GET/api/dashboard/stats全ロールダッシュボード統計（スキル分布・承認待ち件数など） ここで試す
GET/api/dashboard/trend全ロール週次の成長トレンドデータ ここで試す
{{ m.python_sample('import requests BASE_URL = "http://localhost:8190" session = requests.Session() # (1) ログイン（フォーム送信、Cookieはsessionに自動保存される） session.post(f"{BASE_URL}/login", data={ "username": "your_username", "password": "your_password", }) # (2) 認証が必要なAPIを呼び出す（同じsessionなのでCookieが自動送信される） resp = session.get(f"{BASE_URL}/api/dashboard/stats", params={"group_id": 0}) print(resp.json())') }}

Skills

MethodPath必要ロール説明
POST/api/skills/{skill_id}/level全ロール自分のスキルレベルを申告・更新（User権限は承認者必須） ここで試す
GET/api/skills/{skill_id}/panel全ロールスキル詳細パネル（サブスキル・根拠・上書き情報） ここで試す
GET/api/my-skills/{skill_id}/detail全ロール自分のスキル詳細を取得 ここで試す
GET/api/skills/timeline/{target_user_id}全ロール（他人を指定するにはManager以上）成長タイムラインのデータ取得 ここで試す
GET/api/users/{target_user_id}/skill-summaryManager以上指定ユーザーのスキルサマリー（AI要約含む） ここで試す
POST/api/skills/{skill_id}/request-revoke全ロール承認済みスキルの取り消しを申請 ここで試す
POST/api/skills/{skill_id}/cancel-revoke全ロール取り消し申請の取り下げ ここで試す
{{ m.python_sample('import requests BASE_URL = "http://localhost:8190" session = requests.Session() session.post(f"{BASE_URL}/login", data={"username": "...", "password": "..."}) # スキルレベルを更新（Form送信。Adminは即承認、Userは承認待ちになる） resp = session.post(f"{BASE_URL}/api/skills/123/level", data={ "level": 2, "approver_id": 5, # User権限の場合は必須 }) print(resp.json()) # => {"ok": true, "skill_id": 123, "level": 2, ..., "approval_status": "pending"}') }}

他のSkills系エンドポイントもGET/POST + Cookieセッションの同じパターンで呼び出せます。
