業務マップで選択する User Manager Admin

サイドバーの「スキルマップ」をクリックすると、スキルの探し方を2通りから選べる入口ページが表示されます。

入口での選択

{% call m.mockup_browser("skillview.local/skills/start") %} {% endcall %}
  • 直接スキルを選択 — これまでと同じ、カテゴリー・難易度ティアでのスキル一覧（スキルマップ（自己申告）を参照）
  • 業務マップから選択 — 業務・作業内容の単位（「業務エリア」）でグルーピングされた一覧。実際の仕事の流れに沿ってスキルを探せます

業務マップからの申告

業務エリアは階層（大分類 → 中分類 → 個別エリア）になっており、末端のエリアカードにそのエリアに紐づくサブスキルのチェックリストが表示されます。

1エリアカードを開き、できるサブスキルにチェックを入れる
2「申告」ボタンでそのエリア（またはスキル単位）のチェック内容を一括送信
3送信内容は通常のスキル申告と同じ仕組みで処理されます（レベル自動算出 → 承認待ち、Admin/Managerは自動承認）
業務マップは既存のスキル・サブスキルの「見せ方を変えただけ」の入口です。どちらから申告しても、同じスキルデータ・同じ承認フローに反映されます。
業務エリアがグループ限定で設定されている場合、自分が所属するグループに関連するエリアのみ表示されます。
