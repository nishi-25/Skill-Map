ダッシュボード User Manager Admin

スキル状況の概要を確認する画面です。ロールによって表示内容が変わります。Manager/Adminはログイン後、まず「全体モード」のダッシュボードが表示されます（担当グループにスコープされます）。

フィルター

上部のドロップダウンでユーザーまたはグループを選択して絞り込めます。URLにパラメータとして保存されるためブックマークも可能です。表示モードは「全体概要」「グループ」「個人」の3種類です。

表示内容（全体モード）

  • 統計カード（対象メンバー・総申告スキル・平均レベル・カタログ総数）— カーソルを乗せると詳細が表示されます
  • グループ別サマリー — 各グループの平均レベルとクリックでグループビューへ
  • ティア別進捗 — 難易度（初級〜上級）ごとの達成率をリングで表示
  • スキル成長トレンド — 過去12週のレベルUP件数（週次）をグラフ表示
  • メンバーランキング — 平均レベル順のランキング表
  • 成長速度ランキング — 直近30日でレベルアップが多いメンバー
  • 最近の申告・更新 — 直近の変更履歴（Manager/Admin向け）
CSVエクスポートボタンでスキルマトリクスをダウンロードできます。
Admin/Managerの違い: Managerは担当グループにスコープされた集計のみ閲覧できます。Adminは組織全体のメンバー・グループを対象に集計を確認できます。
