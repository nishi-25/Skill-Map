ユースケース: Userの操作フロー User

一般User権限のメンバーが、自分のスキルを申告して承認されるまでの一連の流れを順番に説明します。各機能の詳しい説明はスキルマップ（自己申告）や自分の申請状況のページも参照してください。

{% call m.step_card(1, "ログインする", "/login") %}

発行されたユーザー名・パスワードでログインします。初回ログイン後はロール別のガイドツアーが自動で始まります。

{% call m.mockup_browser("skillview.local/login") %} {{ m.mockup_row("ユーザー名") }} {{ m.mockup_row("パスワード") }} {% endcall %} {% endcall %} {% call m.step_card(2, "ダッシュボードで自分の状況を確認する", "/dashboard") %}

ログイン後、自分のスキル件数・平均レベル・最近の変化が表示されます。まずここで全体像を把握します。

サイドバーの「スキルマップ」から、難易度ティア別またはカテゴリー別にスキル一覧を確認します。
